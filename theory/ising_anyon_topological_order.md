# Ising Anyon Topological Order

Ising anyons realize a topological order described by the Ising modular tensor category (or su(2)_2 Chern-Simons theory).

## Key Features
- Non-abelian statistics from the σ particle.
- Fusion: σ × σ = 1 + ψ
- Appears in theoretical descriptions of ν=5/2 Moore-Read state.
- Braiding generates Clifford gates but is not universal for quantum computation.

## Relation to Codex
Simpler non-abelian layer than Fibonacci. Useful for more physically accessible protected braiding, while Fibonacci represents the full self-similar universal layer.
