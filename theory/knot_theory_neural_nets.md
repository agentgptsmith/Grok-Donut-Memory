# Knot Theory in Neural Nets + Codex Context

Knot theory studies embeddings of circles in 3D space and their invariants.

## Relevance to Neural Nets
- Recurrent / stateful networks can be viewed as having temporal "strands".
- Attractor dynamics in RNNs/Transformers sometimes form looped structures analogous to knots.
- Topological data analysis (TDA) already uses persistent homology on neural activations.

## Connection to Our Codex
We treat flux tubes as strands and memory cycles as closed braids/knots. Jones polynomial-style invariants could serve as topological checksums for important memory patterns — making them verifiable and protected.

This is a natural extension of using braid theory for memory robustness.
